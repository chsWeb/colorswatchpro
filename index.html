Claude Automation Prompts

Check the walkthrough video above for copy & paste prompt templates designed for the Figma MCP workflow. Run them in sequence to build a complete color system from scratch. #claudemate

Prompt 1

Create Color Variables

Create a set of color variables in a new collection called _primitives. The first color is brand-primary, with a 500 value of hsl(245 52 50). Create a second color for neutral with a 500 value of hsl(245 4 50). Create 25, 50, 100–400, then 600–900 and 950 where 25–400 are tints and 600–950 are shades.

Prompt 2

Start Color Swatch Pro

Use the component_color-swatch to create a color swatch for each variable in _primitives. Fill the layer "color" with the variable's color value. _color-swatch has a #hex and HSL color values, please fill that in for each linked variable.

Prompt 3

Color Contrast & Accessibility

Update the color APCA contrast value for each color and set Pass or Fail for Large Text, Fluent Text, Body Text and Preferred Text. The _color-swatch component has a layer named 00, replace that with the APCA value. Toggle each component property true or false based on the APCA value.

Prompt 4

Optimize Contrast

Adjust each swatch for better contrast. The _color-swatch component has a property for dark mode. By default it is in light mode. Update each swatch to dark where it increases the potential use cases for each color. Please update the APCA value and pass/fail values for all swatches changed to Dark Mode.

Prompt 5

Organize

Wrap each color set in a new frame with auto layout applied. Padding 24px and gap 24px. Add a card at the beginning of the color set with a title and description of the color and its use cases.

Prompt 6

Fixes & Tokens

Update the name and color value (##) for each color using the _primitive name.
